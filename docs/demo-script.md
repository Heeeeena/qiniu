# Demo 脚本

建议视频时长控制在 3 到 5 分钟。

1. 说明场景：独立游戏开发者需要快速制作一组风格统一的 2D 素材。
2. 展示工作台：项目名称、风格包、素材需求、类型、风格、尺寸、数量、色板、透明背景、一致性种子。
3. 应用 `Dungeon Pixel Pack`，输入示例：一套地牢探索游戏里的蓝色魔法水晶、入口石门和发光地砖。
4. 点击生成，展示增强后的 Prompt、约束标签和素材预览。
5. 展示质量检查：尺寸规范、透明通道、metadata、Sprite Sheet、风格一致性。
6. 展示单张 PNG 下载。
7. 展示 ZIP、Sprite Sheet、metadata.json 导出。
8. 打开 ZIP 说明导出的 `/images` 和 `metadata.json`。
9. 展示历史记录和风格包复用参数。
10. 说明架构：Vue 前端、FastAPI 后端、Pillow 本地生成器，后续可替换真实图像模型。
11. 总结亮点：生成效率、风格一致性、质量验收、游戏开发流程友好。
